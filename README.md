# Hi, I'm Sebastiaan 👋

Frontend developer (FDND) from the Netherlands. I build with **SvelteKit, Vue, Astro and React**, and I care about accessibility and web standards.

Right now I'm rebuilding frontend at **IntoGolf**, a SaaS used by ~100 golf clubs across the Netherlands.

Besides frontend, I'm a powerlifter and powerlifting coach, which is where **[PrimedLifter](https://primedlifter.app)** came from: I was coaching out of spreadsheets, got fed up, and built the app I wanted instead. It's live, with real coaches and athletes using it.
